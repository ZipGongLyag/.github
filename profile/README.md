## 🗃️ 프로젝트 소개
**집공략**은 공공데이터 포털의 [국토교통부_아파트매매 실거래 상세 자료](https://www.data.go.kr/tcs/dss/selectApiDataDetailView.do?publicDataPk=15057511)와 [Kakao Map API](https://apis.map.kakao.com/web/sample/)를 활용한 주택 매물 검색 서비스입니다.
